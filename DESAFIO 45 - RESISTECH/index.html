Apresentação

Bem-vindo(a) à disciplina de Desenvolvimento Web! Nesta disciplina, você irá aprender os fundamentos do desenvolvimento de aplicações para a web utilizando as mais recentes tecnologias e ferramentas.

Durante o curso, você irá aprender sobre HTML, CSS, JavaScript, React e outras tecnologias essenciais para o desenvolvimento de aplicações web modernas. Além disso, você irá desenvolver projetos práticos para aplicar os conceitos aprendidos e fortalecer suas habilidades de programação.

Estamos animados para trabalhar com você nesta jornada de aprendizado e desenvolvimento!

Conteúdo Programático & Referencial Bibliográfico

  • Conceitos de aplicações web
  • Ambientes de desenvolvimento
  • Sistema de controle de versão e de integração
  • Plataformas de Hospedagens e Containers
  • Protocolo DNS/HTTP/HTTPS
  • Uso de linguagens para desenvolvimento WEB
  • Desenvolvimento de aplicações orientadas a objetos com arquiteturas Cliente-servidor e MVC
  • Segurança de aplicações web
  • Frameworks de desenvolvimento web
